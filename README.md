## 🚀 Featured Projects | 精选项目

### 1. Interstellar Pong | 星际乒乓 🏓
*A multi-sprite physics simulation focusing on vector reflection and concurrent event handling.*
*一个专注于向量反射和并发事件处理的多角色物理模拟项目。*

* **Key Technical Features / 技术亮点**:
    * **Vector Reflection / 向量反射**: Implemented realistic bounce logic using the formula: `180 - direction`.
    * **Velocity Scaling / 速度递增**: Dynamic difficulty management via state variables.
    * **Asynchronous Input / 异步输入**: Multi-player support handling simultaneous key-press events.
* **CS Concepts**: Game Loop, Variables, Collision Detection, Parallelism.
* **CS 概念**: 游戏循环、变量、碰撞检测、并行处理。

## 💡 Pedagogy & Philosophy | 教学理念与思考

As an engineer (B.E. in Software Engineering), I believe in **"De-blackboxing"** coding for kids. My projects focus on:
作为一名软件工程学士，我坚信要为孩子们**“拆解编程黑盒”**。我的项目侧重于：

1. **Industrial Logic (工业级逻辑)**: Moving beyond "move 10 steps" to understand coordinate systems and physics vectors.
   超越“移动10步”，转而理解坐标系和物理向量。
2. **Clean Architecture (整洁架构)**: Even in Scratch, I emphasize script organization, message broadcasting, and decoupling.
   即使在 Scratch 中，我也强调脚本组织、消息广播和解耦。
3. **Bridge to Text-based Coding (通往文本编程的桥梁)**: Mapping Scratch blocks to Python/C++ concepts to prepare students for the next level.
   将 Scratch 积木映射到 Python/C++ 概念，为学生进阶做好准备。

## 🕹️ How to Run | 如何运行

To view or edit these projects, follow these steps:
要查看或编辑这些项目，请按照以下步骤操作：

1. **Download**: Navigate to the `projects/` folder and download the `.sb3` file (e.g., `pongFinished.sb3`).
   **下载**: 进入 `projects/` 文件夹并下载 `.sb3` 文件（例如 `pongFinished.sb3`）。
2. **Open Scratch**: Visit the [Scratch Online Editor](https://scratch.mit.edu/projects/editor/).
   **打开 Scratch**: 访问 [Scratch 在线编辑器](https://scratch.mit.edu/projects/editor/)。
3. **Load**: Click on **File** -> **Load from your computer** and select the downloaded file.
   **加载**: 点击 **文件 (File)** -> **从电脑上传 (Load from your computer)** 并选择该文件。
